Algorithms

CatBoost Algorithm

CatBoost is a high-performance open source library for machine learning (gradient boosting) on decision trees, its name coined from “Category” and “Boosting.” It was developed by Yandex (Russian Google ) in 2017. According to Yandex, CatBoost has been applied to a wide range of areas such as recommendation systems, search ranking, self-driving cars, forecasting, and virtual assistants.

Why CatBoost Algorithm?

Fast and scalable GPU version

Train your model on a fast implementation of gradient-boosting algorithm for GPU. Use a multi-card configuration for large datasets.

Overfitting Detector

CatBoost has an overfitting detector that can stop training earlier than the training parameters dictate if overfitting occurs. CatBoost implements overfitting detection using two strategies:Iter, IncToDec

Cross-validation

CatBoost allows to perform cross-validation on the given dataset. In cross-validation mode, the training data is split into folds of learning and evaluation.

Missing Value Support

CatBoost provides three inherent missing values strategies for processing missing values. “Forbidden”:Missing values are interpreted as an error as they are not supported, “Min” , “Max”.

Improved accuracy

Reduce overfitting when constructing your models with a novel gradient-boosting scheme.

Great quality without parameter tuning

Reduce time spent on parameter tuning, because CatBoost provides great results with default parameters

Categorical features support

Improve your training results with CatBoost that allows you to use non-numeric factors, instead of having to pre-process your data or spend time and effort turning it to numbers.

Fast prediction

Apply your trained model quickly and efficiently even to latency-critical tasks using CatBoost's model applier

Modules

Visual Metadata Module

By analyzing different physical parameters like total area in Sqft, slab area in Sqft , building age in Years, floor numbers, No of BHK, No of bathrooms, house type, main door facing, type of furnish,availability of balcony, availability of parking and availability of dining room . By using this parameters we estimate price of the house.

Amenities Locator Module

Feature will access the location of the house by using Google Maps API and mark all the public amenities available nearer to house into 5 levels such as public amenities under 500 meters as level 1, under 1000 meters as level 2, under 1500 meters comes as level 3, under 2000 meters as level 4, under 2500 meters as level 5. We are extending the feature by adding distance, direction from house to particular public amenity. The distance plays vital role for evaluating prices more accurately.
